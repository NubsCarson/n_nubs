## Getting Started

Requirements:
- Node.js & npm - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)
- Solana CLI tools (optional)

```sh
# Clone and setup
git clone https://github.com/NubsCarson/n_nubs.git
cd n_nubs

# Install dependencies
npm i

# Start development server
npm run dev
```
